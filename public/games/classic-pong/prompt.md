# Prompt Used to Generate Classic Pong

```
Create a classic Pong game with the following requirements:
- 800x500 canvas with black background
- White paddles on left and right sides (10px x 100px)
- White ball that bounces off paddles and walls
- Left paddle controlled by W and S keys
- Right paddle controlled by simple AI that follows the ball
- Score display showing both player and AI score
- Ball resets to center after each point
- First to 5 points wins
- Display winning message when someone reaches 5 points
- Add restart button after game ends
- Make it visually appealing with gradients and animations
- Add center line to divide the playing field
- Ball should speed up slightly when hitting paddles
```

## AI Model Output

The AI successfully generated a complete, playable Pong game with:

- Smooth paddle movement and collision detection
- AI opponent that tracks the ball with slight imperfection
- Score tracking and win condition
- Visual polish with gradients, shadows, and animations
- Responsive controls with keyboard input
- Clean, retro aesthetic
