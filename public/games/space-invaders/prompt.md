# Prompt Used to Generate Space Invaders

```
Create a classic Space Invaders game with these requirements:
- 800x600 canvas with black background
- Player spaceship at bottom that moves left/right with arrow keys
- Shoot bullets with spacebar
- 5 rows x 10 columns of red invaders at the top
- Invaders move horizontally and down when hitting edges
- Invaders speed up as they get hit
- Game ends if invaders reach the player or bottom
- Player wins when all invaders are destroyed
- Display score that increases when hitting invaders
- Display remaining lives (3 lives)
- Animated game over and win screens
- Smooth movement and collision detection
- Visual feedback with colors (green player, yellow bullets, red invaders)
- Restart button after game ends
```

## AI Model Output

The AI generated a complete Space Invaders clone with:

- Grid-based invader formation (5x10)
- Dynamic invader movement with edge detection
- Speed progression as game progresses
- Bullet collision detection with invaders
- Score tracking system
- Win/loss conditions with animated messages
- Visual distinction between player, bullets, and invaders
- Smooth controls with keyboard input
- Proper game state management
