```

- A 400x400 canvas with a light background
- Red squares (30x30px) that appear at random positions
- Click detection - when player clicks a square, score increases by 1
- Display current score prominently
- Simple, clean styling with a gradient background
- No external libraries or dependencies
- Include basic instructions for the player
- Use modern JavaScript with canvas API
```
