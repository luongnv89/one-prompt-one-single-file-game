# Prompt Used to Generate Memory Match Game

```
Create a memory matching card game with these features:
- Grid of face-down cards (4x4 or 6x4 based on difficulty)
- Each pair has matching emoji or symbol
- Cards flip when clicked (3D flip animation)
- Player must find all matching pairs
- Show moves counter and timer
- Show progress (pairs found out of total)
- Game ends when all pairs are found
- Display congratulations message with final stats
- Difficulty selector for different grid sizes
- Use cute animal emojis as card faces
- Card backs show "?" mark
- Matched cards stay flipped and turn green
- Smooth animations and visual feedback
- Reset/New Game button to start over
- Responsive card grid layout
```

## AI Model Output

The AI generated a complete memory matching game with:

- Dynamic grid generation based on difficulty
- 3D flip animations using CSS transforms
- Shuffle algorithm for random card arrangement
- Win detection with statistics display
- Multiple difficulty levels (4x4, 6x4)
- Visual feedback for matched cards
- Timer and move counter
- Emoji-based card faces for visual appeal
- Smooth transitions and animations
